# OpenPRISM — Public Releases

Pre-built APK releases for **OpenPRISM**, an open-source Android app for Smart Glasses.

## Download

| Version | APK | Notes |
|---|---|---|
| **v0.2.1** | [OpenPRISM-v0.2.1.apk](OpenPRISM-v0.2.1.apk) | Bugfix: all feature cards visible on home screen |
| v0.2.0 | [OpenPRISM-v0.2.0.apk](OpenPRISM-v0.2.0.apk) | Dark theme with prismatic UI, live video streaming, photo capture, multi-platform streaming |

## Install

1. Download the APK to your Android device
2. Open it and allow installation from unknown sources if prompted
3. Requires Android 12+ (API 31), arm64 device

## Requirements

- Meta Ray-Ban Smart Glasses (any model with camera)
- Meta companion app installed, logged in, and glasses paired
- Developer Mode enabled (Meta companion app → Settings → App Info → tap version 5x → enable)
